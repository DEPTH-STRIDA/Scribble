Техническое задание на разработку веб-приложения для организации каталога статей

1. Цель проекта:
   Реализовать веб-приложение для создания, редактирования и оценки статей.

2. Основные функции:
   - Создание новых статей.
   - Редактирование существующих статей.
   - Группировка статей по категориям.
   - Возможность оценивать статьи (лайк).
   - Просмотр статей в ленте + возможность фильтрации по категориям (доп фильтры по типу автора, дате публикации и тд)
   - Администратор может удалять и редактировать статьи и пользователей

3. Требования к функционалу:
   - Авторизация пользователей с ролями (пользователь и администратор).
   - Пользователь может редактировать/удалять только свои статьи.
   - Администратор может редактировать/удалять любые статьи.

4. Технологии:
   - База данных: PostgreSQL с использованием ORM GORM.
   - Бэкенд: Go.
   - Фронтенд: React с использованием Tailwind CSS и Vite.

5. Шаблоны:
   - Один шаблон в Go для HTML-дизайна сообщения с кодом на почту.
   - Остальная верстка будет реализована на React.

6. Дизайн:
   - Дизайн не требуется, можно использовать Bootstrap или Tailwind CSS.

7. Сроки:
   - Срок выполнения: 2 дня.

8. Дополнительные комментарии:
   - Оценка статей будет осуществляться в виде лайка.
   - Возможность проявления фантазии для ускорения реализации.
